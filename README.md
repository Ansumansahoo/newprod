# ChainMed 🏥⛓️

## Decentralized Medical Supply Chain Tracker

A fully functional **Blockchain + Java + React** application tracking pharmaceuticals on the blockchain.

### 🏗️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | React 18 + Vite + TailwindCSS + Web3.js |
| **Backend** | Java 17 + Spring Boot 3 + REST API |
| **Database** | PostgreSQL (open source) |
| **Blockchain** | Ethereum + Solidity Smart Contracts |
| **Dev Tools** | Hardhat, Ethers.js |

### 📁 Project Structure

```
newprod/
├── frontend/          # React + Vite + TailwindCSS
├── backend/           # Spring Boot Java 17 REST API
├── blockchain/        # Solidity contracts + Hardhat
├── docker-compose.yml # Full stack with PostgreSQL
└── README.md
```
